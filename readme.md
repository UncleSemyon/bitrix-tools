# Bitrix Tools

Классы-помощники для разработки под 1С-Битрикс

## Начало работы

### Необходимые условия окружения

Для запуска проекта в системе должны быть установлены:

- Composer
- PHP

### Установка


```bash
composer require spaceonfire/bitrix-tools
```
